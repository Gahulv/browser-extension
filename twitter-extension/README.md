# Twitter Comment Filter

一个可直接加载到浏览器的 `Manifest V3` 插件，用于在 `x.com` / `twitter.com` 推文详情页按自定义文本或正则规则移除评论。

## 功能

- 自定义添加多条屏蔽文本
- 支持普通文本规则和正则表达式规则
- 普通文本会忽略大小写、空格、常见标点和链接
- 自动监听评论区动态加载内容

## 使用方式

1. 打开 Chrome 或 Edge 的扩展管理页。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选中当前目录：`twitter-extension`
5. 打开任意 `x.com/.../status/...` 或 `twitter.com/.../status/...` 页面。
6. 点击插件图标，添加你要屏蔽的文本或正则规则。

## 匹配说明

- 普通文本规则：去掉空格、标点、链接后做直接包含匹配
- 正则规则：使用 `/pattern/flags` 格式，按评论原文匹配
- 当前实现默认只在推文详情页处理，避免误伤首页时间线
