# Weibo Hot Search Filter

一个可直接加载到浏览器的 `Manifest V3` 插件，用于按自定义文本屏蔽微博热搜，并把命中项直接从列表移除。

## 功能

- 自定义添加多条屏蔽文本
- 文本匹配忽略大小写、空格和常见符号
- 支持规则导出与导入（可跨设备迁移）
- 支持通过 GitHub raw URL 同步云端规则
- 本地规则和云端规则会同时参与匹配
- 热搜列表动态变化时自动重扫并移除命中项

## 使用方式

1. 打开 Chrome 或 Edge 的扩展管理页。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择目录：`weibo-extension`
5. 打开微博热搜页：`https://s.weibo.com/top/summary`
6. 点击插件图标，添加你要屏蔽的文本。

## 云端规则

1. 将导出的规则 JSON 文件上传到 GitHub 仓库。
2. 在 GitHub 上获取该文件的 raw URL（`https://raw.githubusercontent.com/...`）。
3. 将 raw URL 填入插件的”GitHub Raw URL”输入框，点击”同步云端规则”。

规则文件所在仓库应为公开仓库，或 raw URL 必须能在浏览器里无鉴权访问。

规则 JSON 格式：

```json
{
  "exportedAt": "2026-04-30T00:00:00.000Z",
  "patterns": ["广告", "示例名称"]
}
```

也支持纯文本格式，每行一条规则。

## 匹配说明

- 会先尝试在原始文本中直接包含匹配
- 同时使用规范化文本匹配，增强对空格和符号变化的兼容
- 本地规则和已缓存的云端规则合并后匹配；本地删除不会删除云端规则，需要修改 GitHub 文件后重新同步
